\chapter*{\hfill Abstract \hfill}

This thesis presents the design and development of \textbf{HayaBook}, an on-demand service booking mobile platform targeting the Algerian market. The platform connects clients with service providers (barbers, doctors, tutors, etc.) through a Flutter mobile application, a React administration dashboard, and a Node.js/Express backend server with a PostgreSQL database managed via the Prisma ORM. The system integrates JWT-based secure authentication, real-time communication via Socket.io for notifications and instant messaging, and a full booking engine with availability verification and automatic status transitions. The obtained results demonstrate the feasibility of a complete, scalable, and secure digital solution for on-demand service management in Algeria.

\textbf{Keywords:} Online booking, Mobile application, Flutter, Node.js, Prisma, PostgreSQL, Socket.io, REST API, JWT, Real-time.
